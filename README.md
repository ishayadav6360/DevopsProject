# Gym Fitness App

## Overview
Simple Gym Fitness Web Application with Login and Signup using Node.js.

## Tools Used
- HTML
- CSS
- Node.js
- Express.js
- Docker
- Jenkins
- GitHub

## Features
- User Signup
- User Login
- In-memory user storage
- Dockerized application
- CI/CD pipeline using Jenkins

## Run Project

npm install
npm start

## Docker Commands

docker build -t gym-fitness-app .
docker run -p 3000:3000 gym-fitness-app

## CI/CD Workflow
GitHub → Jenkins → Docker → Deployment
